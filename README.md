# deno-hello

## 実行

```shell
deno run hello.ts
```

## コンパイル

- ホストPCでコンパイル
    ```shell
    deno compile --output target/ hello.ts
    ```
- windows
    ```shell
    deno compile --output target/ --target x86_64-pc-windows-msvc hello.ts
    ```
